# wasm-assemblyscript-example

WebAssembly example using [AssemblyScript](https://www.assemblyscript.org).

```bash
npm install -g assemblyscript
```

```bash
asc hello-world.ts -o hello-world.wasm
```

```bash
python -m http.server 8000
```
